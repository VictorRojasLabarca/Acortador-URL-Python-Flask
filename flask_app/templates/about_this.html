About this | Sobre esto

English

Hello, I hope you're doing well. In this section, you'll find information about the project you're viewing, including the technologies used, the GitHub repository, and some important details worth mentioning.

Let's start with the link to the GitHub repository. There, you'll find most of the source code—I say 'most' because you'll need a '.env' file that I didn't upload for security reasons. This '.env' file contains the environment variable, specifically the 'secretkey' used by Flask to sign sessions. It's easy to create and should be placed outside the 'flask_app' folder, in case you want to use the URL shortener for personal purposes. That should be enough to get it up and running.

Here's what was used in this URL shortener:

  • • Css | BootStrap.
  • • Python | Flask.
  • • Python | Pipenv.
  • • Python | Dotenv.
  • • Python | Pyshorteners.

PS: The project is in English because I'm practicing the language.
